## Описание SpringBootApplication
Аннотация `@SpringBootApplication` объединяет три аннотации:
- `@Configuration` – указывает, что класс содержит настройки Spring.
- `@EnableAutoConfiguration` – автоматически настраивает приложение.
- `@ComponentScan` – выполняет сканирование пакетов на наличие аннотированных компонентов.

## Дополнительные параметры:
- `exclude` - Исключите определенные классы автоконфигурации, чтобы они никогда не применялись.
- `excludeName` - Исключите определенные имена классов автоконфигурации, чтобы они никогда не применялись.
- `scanBasePackages` - Базовые пакеты для сканирования аннотированных компонентов. Используйте scanBasePackageClasses() для типобезопасной альтернативы строковым именам пакетов.
- `scanBasePackageClasses` - Типобезопасная альтернатива для scanBasePackages()указания пакетов для сканирования аннотированных компонентов. Пакет каждого указанного класса будет просканирован.
- `nameGenerator` - Класс BeanNameGenerator, который будет использоваться для именования обнаруженных компонентов в контейнере Spring.
- `proxyBeanMethods` - отключает или подключает проксирвоание бинов.
